# User Management System - Code Challenge - Front-end

## Dev

### Install dependencies with Yarn

```bash
yarn install
```

### Run the application

Make sure it's using the 3001 port.

```bash
yarn dev
```

Also make sure that the backend is up and running, otherwise all requests will
fail.
